## **Array Shift**
---

### Problem Domain

Given an array insert the input integer value to the given array in the middle of the index.

---

### Inputs and Expected Outputs

| Input | Expected Output |
| :----------- | :----------- |
| [2,4,6,8], 5 | [2,4,5,6,8] |
| [4,8,15,23,42], 16 | [4,8,15,16,23,42] |


---

### Big O


| Time | Space |
| :----------- | :----------- |
| O(n) | O(n) |
