# **FIFO Animal Shelter**

**Author: Jin Kim**

---

### Problem Domain

Write a method call `Enqueue` that adds `animal` to the shelter. `animal` can be either dog or cat object. 

---

### Inputs and Expected Outputs

| Input | Arg |Expected Output |
| :----------- |:--------- | :----------- |
| []| Dog | [Dog]|

 
 
---

### Big O


| Time | Space |
| :----------- | :----------- |
| O(2n) | O(n) |
