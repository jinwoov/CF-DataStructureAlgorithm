# **Sum Of All Number**

**Author: Jin Kim**

---

### Problem Domain

Write a method that will take in a tree and traverse through to find any odd number, then output the sum of all odd numbers.

---

### Inputs and Expected Outputs

#### Stack

             1
          2      3
       4    200  6    7

|Output |
| :--------- | 
| 11 |

---

### Big O


| Time | Space |
| :----------- | :----------- |
| O(n) | O(h) |
